Structure

The boxes at the bottom are the original array. Above them, every node stores the sum of its interval.

Algorithm Array: `tree[]`

Internally, the segment tree is stored in an array. For a node at index i, the left child is 2*i + 1 and the right child is 2*i + 2. During build, these positions are filled from the leaves back up to the root.
